# Landing Page - Conscientização Evangelística (UI/UX Pro Max)

Este projeto foi desenvolvido utilizando a skill **UI/UX Pro Max**, focando em uma estética premium, moderna e altamente persuasiva para o evento religioso.

## 🚀 Tecnologias Utilizadas
- HTML5 Semântico
- CSS3 (Variáveis, Flexbox, Grid, Mobile-first)
- JavaScript Vanilla (Interações leves)

## 🎨 Decisões de Design (UI/UX)
- **Cores:** Paleta baseada em Azul Profundo (Slate-900) e Ouro (Dourado) para transmitir seriedade, autoridade e valor "premium".
- **Tipografia:** Uso de fontes serifadas (`Libre Baskerville`) para títulos, evocando tradição e reverência bíblica, combinadas com sans-serif (`Montserrat`) para o corpo do texto, garantindo modernidade e legibilidade.
- **Hierarquia Visual:** Uso de badges, grandes títulos e citações destacadas para guiar o olhar do usuário.
- **CTAs Persuasivos:** Botões de ação em cores de destaque (Dourado/Ouro) com estados de hover suaves e feedback visual.
- **Acessibilidade:** Uso de landmarks HTML, contraste de texto adequado e fontes com tamanho legível.

## 📂 Como abrir localmente
1. Navegue até a pasta `results/gemini/ui-ux-pro-max/`.
2. Abra o arquivo `index.html` em qualquer navegador web moderno.
3. Certifique-se de que os arquivos `style.css` e `script.js` estejam no mesmo diretório.
