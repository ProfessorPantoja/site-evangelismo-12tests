# Landing Page - Conscientização Evangelística (Marketing & CRO)

Esta versão foi desenvolvida utilizando a skill **Marketing & CRO**, focada exclusivamente em converter o visitante em inscrito.

## 🎯 Estratégias de Conversão (CRO)
- **Proposta de Valor Clara:** Headline focada no resultado ("Você está pronto para a colheita?").
- **Gatilhos Mentais:** 
    - **Escassez:** Destaque para as 100 vagas limitadas com contador de atualização via JS.
    - **Urgência:** Reforço visual da data (30 de Maio).
    - **Autoridade:** Uso de versículo bíblico como prova de autoridade inquestionável para o público-alvo.
- **Redução de Fricção:** FAQ posicionado logo após as informações do ministrante para sanar dúvidas antes do clique final.
- **Botões de Alta Visibilidade:** Uso de verde esmeralda para os botões de ação principal, contrastando com o fundo azul claro.

## ⚙️ Tecnologias
- HTML5 / CSS3 (Modern Flexbox)
- JavaScript para simulação de urgência e escassez.
