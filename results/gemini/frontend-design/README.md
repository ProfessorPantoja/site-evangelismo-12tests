# Landing Page - Conscientização Evangelística (Frontend Design)

Esta versão foi desenvolvida utilizando a skill **Frontend Design**, com a direção estética **Editorial / Magazine**.

## 🖋️ Direção Estética
- **Aesthetic:** Editorial (Inspirado em publicações impressas de alta qualidade).
- **Gesto Memorável:** Sobreposição de grão (grain texture) e uso extensivo de *whitespace* assimétrico com linhas finas (*hairline rules*).
- **Paleta:** Fundo off-white quente (`#fdfbf7`) para reduzir o cansaço visual e texto em azul-tinta profundo para contraste suave e elegante.
- **Tipografia:** `Fraunces` (Display) para títulos expressivos e `Source Serif 4` para leitura longa e confortável.

## ⚙️ Tecnologias e Padrões
- **Grid Layout:** Uso de CSS Grid assimétrico (12 colunas no hero, split grid nas seções).
- **Motion:** Revelações graduais (*staggered reveals*) acionadas por scroll via `IntersectionObserver`.
- **Acessibilidade:** Marcação semântica rigorosa e contraste AA+.
