# Landing Page - Conscientizacao Evangelistica

## Arquivos
- `index.html`
- `style.css`
- `script.js`

## O que foi implementado
- Landing page premium em HTML/CSS/JS puro.
- Estrutura mobile-first e responsiva.
- Secoes: Hero, Sobre o Evento, Informacoes do Evento, Como Chegar (com mapa iframe), Ministrante e Conteudo, Inscricao/Vagas/Pagamento, FAQ e Rodape.
- CTA `Inscreva-se` com o link oficial no topo, no meio e no final da pagina.
- Acessibilidade base: landmarks semanticos, skip link, foco visivel, contraste e menu com atributos ARIA.
- Todos os campos pendentes com `FALTA: ...` foram expostos no conteudo.

## Como abrir localmente
1. Entre na pasta `results/codex/frontend-design/`.
2. Abra `index.html` no navegador.
