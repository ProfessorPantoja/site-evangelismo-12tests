# Landing Page - CONSCIENTIZACAO EVANGELISTICA

Arquivos desta entrega:
- `index.html`
- `style.css`
- `script.js`

## O que foi implementado
- Landing page premium e moderna com foco em conversao para inscricao.
- Estrutura mobile-first e responsiva.
- Acessibilidade: landmarks, foco visivel, labels em campos, FAQ com `aria-expanded`, link de pulo de conteudo.
- CTA `Inscreva-se` no topo, no meio e no final, apontando para:
  - `https://bit.ly/admpf-evento-evangelismo-2026`
- Secoes obrigatorias:
  - Hero
  - Sobre o Evento
  - Informacoes do Evento
  - Como Chegar (com mapa iframe)
  - Ministrante e Conteudo
  - Inscricao/Vagas/Pagamento
  - FAQ
  - Rodape
- Todos os campos faltantes foram exibidos como `FALTA: ...`.

## Como abrir localmente
Opcao 1 (direto no navegador):
1. Abra `results/codex/ui-ux-pro-max/index.html`.

Opcao 2 (servidor local simples):
1. No terminal, na raiz do projeto, execute:
   ```bash
   cd results/codex/ui-ux-pro-max
   python3 -m http.server 8080
   ```
2. Acesse: `http://localhost:8080`
