# Template Reutilizavel

Use este arquivo como ponto de partida para novas pecas.

## Bloco 1: GUIA

```text
Crie um infografico vertical, em formato retrato, com visual [estilo desejado], sobre [nome do tema].

Tema central:
"[frase-tema]"

Direcao criativa:
- estetica [seria / editorial / didatica / solene / evangelistica]
- fundo [cores principais]
- destaques em [cores secundarias]
- tipografia [nobre / limpa / forte / legivel]
- composicao [cards / colunas / blocos / trilha vertical]
- icones relacionados a [lista de elementos]
- transmitir [mensagem central]
- evitar [o que nao deve aparecer]

Estrutura desejada:
- titulo forte
- subtitulo curto
- introducao breve
- [numero] blocos principais
- fechamento com [nome / cargo / foco]
```
